# 阿里云 Function Compute Api 执行组件


## 前言

通过本组件，您可以无配置的执行 Function Compute 的相关api

## 使用

### 最简使用方法

模版拉取：

```
s cli s-fc -d  // 查看api 内容及调用方法
```

```
s cli s-fc listServices -a default -r cn-hangzhou // 查询Function Compute服务列表， 指定默认秘钥， 指定查询 region
```
